# Roadmap

| Versión | Fecha | Estado | Autor |
|---|---|---|---|
| 0.1.0 | 2026-07-03 | Pendiente | Rodrigo Zanin |

## Propósito

Este documento consolidará el roadmap de versiones de EnergIA (v1 a v4). Actualmente el roadmap del producto vive en PRODUCT_VISION.md, sección "Roadmap del Producto", y este documento tiene como propósito reunir esa información junto con el detalle técnico de planificación (dependencias entre versiones, criterios de salida por versión, riesgos de cada etapa) en un único lugar de referencia para el equipo.

## Contenido previsto

- Referencia y enlace a PRODUCT_VISION.md, sección "Roadmap del Producto", como fuente actual del roadmap de alto nivel.
- Detalle de alcance de Versión 1.0 (MVP): procesamiento de lotes, detección de anomalías, IRE, dashboard, planificador de inspecciones, integración con RRHH.
- Detalle de alcance de Versión 2.0: explicabilidad avanzada, modelos supervisados, feedback automático, optimización de cuadrillas.
- Detalle de alcance de Versión 3.0: predicción de pérdidas, detección de medidores defectuosos, recomendaciones inteligentes.
- Detalle de alcance de Versión 4.0: predicción de demanda, integración GIS, modelos predictivos de mantenimiento.
- Dependencias y criterios de salida entre versiones.
- Riesgos y supuestos por etapa del roadmap.
- Plan de consolidación final de este documento con PRODUCT_VISION.md para evitar duplicación de fuentes.
